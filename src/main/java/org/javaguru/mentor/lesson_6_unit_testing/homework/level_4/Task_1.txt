Написать автоматические тесты на класс FactorialForLoop
в котором реализован метод вычисления факториала.

Класс FactorialForLoop лежит рядом с этим файлом задания.

Для выполнения задания скопируйте класс FactorialForLoop к себе в пакет.

Создайте класс FactorialForLoopTest и добавьте в него
необходимые тестовые методы.

Цели:
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.

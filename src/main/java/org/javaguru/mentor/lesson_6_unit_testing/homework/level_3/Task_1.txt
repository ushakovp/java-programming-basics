Написать автоматические тесты на класс SquareEquation
в котором реализованы методы расчёта корней квадратного уравнения.
Класс SquareEquation лежит рядом с этим файлом задания.

Для выполнения задания скопируйте класс SquareEquation к себе в пакет.

Создайте класс SquareEquationTest и добавьте в него
необходимые тестовые методы.

PS: все public методы должны быть протестированы отдельно друг от друга!

Цели:
- изучить сравнение типов данных double при написании тестов;
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
